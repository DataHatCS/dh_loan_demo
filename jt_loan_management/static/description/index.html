Overview

The Loan Management App is a flexible and powerful app that manages invoices, journal entries, outstanding balances, interest and principal balances, payments, and much more of a loan. Additionally, it offers different ways to pay loan installments and the ability to change installment dates.

Watch On Youtube

Features

Some of the main features of this module are:

You can differentiate loan products by adding the 'Property' category.

Invoices for loan installments are automatically generated by the scheduler based on dates.

Print a PDF report to see loan details.

The user can send an email with a PDF file for loan details.

Server Actions 1. To update loan installments invoices number. 2. To Create loan installment invoices manually before today date.

Selecting loan installment items You can delay installments by a certain number of days or months, and reset them if necessary.

It provides all transaction information such as Transaction history, Payment history, Outstanding balance, Invoices, Due days, Taxes, insurance, principal, and interest in one view.

Creates Processing fee invoice when validating loan. .

The extra loan amount can be paid with some options. For example, cover that amount only with the principal or with installments or both.

If there are any partial or unpaid loan installment invoices, the loan should not close. Closing a loan cancels all draft and open installment invoices. Add ‘C’ after the loan name.

Views loan details and payment amounts from customer view, so the user can see the total loan amount and total payment amount of any customer.

The Total Cashed and Paid details from the product view are shown.

History of loan installment invoices and payments with current balance. Creates a transaction history when the loan installment invoice is validated, and also removes it when it is cancelled.

Payment history of all loan payments and linked invoices with payments. Creates a payment history when you pay and removes it when you cancel the payment. On unreconciliation of payments, removes invoice link from payment history.

Postpone Installments
- Allow to postpone loan installments date by certain number of days or month by selecting specific date.
- This functionality postpone open loan invoices due date and invoice date or postpone installments date which are after selected date.
- By certain month option user has choice to postpone whole invoice or want to postpone invoice by selecting number of items(like Principal, Interest, Tax or Insurance).
- Also provides reset functionaly that will reset all instllments dates and invoices.
- Adds history for all intallment changes.




Extra Payment
- Allow to pay loan amount in different way like,
1. Settle with next due instalment rest with principal
2. Cover all next due installments
3. Cover only principal outstanding
1. Settle with next due instalment rest with principal
- Using this way user can pay first installment and if amount greater than first installment then rest pays in principal amount.
2. Cover all next due installments
- Cover all next due installments option pays all amount in installments(Loan invoices).
3. Cover only principal outstanding
- To pay only in principal amount.
- This also changes installment schedule based on payment amount and payment method.
- Also provide functionality to close loan, this will calculate total open invoices amount plus remaining principal. So user can pay all remaininging amount and close loan directly.
